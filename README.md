# event-service

## Назначение:

### Сервис отвечает за:

- Управление событиями
- Регистрацию пользователей на события
- Публикацию аналитических событий

### Функциональные требования

- Получение списка событий
- Получение одного события по ID
- Регистрация пользователя на событие
- Публикация события регистрации в NATS

## Требования к запуску:
- Docker
- Git

### 1. Клонирование репозитория
`git clone github.com/Telegram-bot-for-register-on-events/event-service`
`cd event-service`

### 2. Cоздание .env
В корне проекта выполните команду:
`mv .envexample .env`.

### 3. Запуск микросервиса
В корне проекта выполните команду:
`docker compose up -d --build`
